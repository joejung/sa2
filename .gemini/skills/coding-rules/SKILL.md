# Coding Rules Skill

This skill enforces specific coding standards for the project.

## Mandates
1. **File Length Limit:** Never exceed 1500 lines per file. If a file approaches this limit, refactor and split it into smaller, modular components.
2. **Logging:** Always add comprehensive logging to functions, especially for error handling, state transitions, and major operations. Use Python's `logging` module.

## How to use
- Activate this skill when writing or refactoring code to ensure compliance.
- When performing code reviews, check for these two specific rules.
